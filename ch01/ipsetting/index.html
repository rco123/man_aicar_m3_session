자동차 IP 설정

인공지능자동차를 네트워크를 통한 연결이 가능하도록 IP 설정과정을 수행한다.

USB를 이용한 설정

처음 인공지능 자동차를 부팅하는 경우, 공유기와의 연결된 이력이 없어서, 인터넷 망에 연결이 되지 않는다.

자동차를 인터넷 공유기와 연결하기 위해서는 다음과 같은 과정을 따른다.

가상 USB 이더넷 어댑터 기능을 통해 Jetson Board는 PC와 Point-to-Point 네트워크를 연결할 수 있습니다.

Jetson Board의 Default USB 고정 IP는 ”192.168.55.1“ 입니다.

Jetson Board외 연결되는 PC의 IP를 192.168.55.100 또는 DHCP(자동주소받음)로 설정하고 가상 로컬 이더넷 네트워크를 형성합니다. 따라서 Jetson nano 보드의 계정 (사용 이름:jetson 및 암호 사용:1111)을 생성한 경우, 네트워크 Jetson 보드의 접속 IP를 192.168.55.1을 지정하여 PC에서 SSH를 이용하여 Jetson Board로 로그인 할 수 있습니다.

따라서 어느 쪽이든 Micro USB 케이블로 Jetson의 터미널에 액세스할 수 있습니다.

항목 내용
login ID jetson
passwd 1111
ip address 192.168.55.1

192.168.55.1로 연결을 한후, MobaXterm 터미널 프로그램을 이용한여, 자동차에 ssh 접속을 수행한다.
