이미지 수집

분류를 해야되는 이미지 디렉토리를 생성하고 해당 디렉토리에 이미지를 수집한다.

이미지 디렉토리는 이름의 사용자가 임의로 정하면 된다.

여기서는 다음과 같은 이름으로 설정하였다.

dol 디렉토리

지정된 디렉토리 안에, 아래와 같은 56장의 이미지를, 정지를 위한 목적으로 해당 이미지를 수집한다.

hsp 디렉토리

지정된 디렉토리 안에, 아래와 같은 이미지를, 가속주행을 목적으로 해당 이미지를 수집한다.

lsp 디렉토리

지정된 디렉토리 안에, 아래와 같은 이미지를, 저속주행을 목적으로 해당 이미지를 수집한다.

road 디렉토리

일반 도로 이미지를 수집하여 넣는다. 일반적으로 주행중 만나는 이미지를 수집한다.

start 디렉토리

출발 지점이 있는 지역의 이미지를 수집하여 넣어둔다.

stop 디렉토리

정지가 필요한 곳의 이미지를 수집하여 넣어둔다.
