  • 2. 자동차 자율주행
  • 2.1 도로이미지 수집

도로이미지 수집

도로주행 학습을 위한 주행도로 이미지 수집및 제작을 위하여 프로그램을 수행한다.

iblock 프로그램 실행

iBock 프로그램을 수행하여 핸드폰 프로그램과 연동하여 이미지 데이터를 저장한다.

CODE -> lane -> 01.net_imgs.xml 블록을 작업화면에 가져와 클릭한다.

프로그램 RUN

프로그램 RUN 버턴을 클릭하여, 프로그램이 인공지능 자동차에서 동작하도록 한다.

핸드폰 프로그램 동작

핸드폰 제어 프로그램을 동작하여 주행도로을 주행하면서 이미지를 생성한다. 진행 방향을 터치 하여 방향을 지정한다.
