색상 모델생성

신호등 색상을 구분을 위한 모델 생성은 다음과 같이 수행하여 모델을 생성한다.

CODE -> light -> 02.train_class.xml

모델 색상 구분을 위한 이미지 수집을 따로 하지 않는다. 신호등 색상 구분은 기 제공된 모델을 그냥 사용하여 색상을 구분하도록 구성되어 있다.
