데이터 학습

준비된 데이터를 이용하여 목적물 디텍션을 위한 학습을 진행한다.

학습과정진행

주어진 데이터를 이용하여 학습을 진행한다.

CODE -> dir -> 01.train.xml

를 호출하여 학습을 진행한다.

학습 진행

  • 데이터가 있는 디렉토리 설정
  • 생성해야 되는 모델 이름 설정
  • 학습 에폭(학습량) 을 설정

RUN 버턴을 클릭하여 학습 과정을 진행한다.
